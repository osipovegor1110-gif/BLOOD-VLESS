ИНСТРУКЦИИ ПО НАСТРОЙКЕ

01. ANDROID

Рекомендуемое приложение: v2rayTun/Happ

  • • Получите ключ в нашем боте.
  • • Нажмите на иконку ➕ в приложении.
  • • Выберите "Import config from clipboard".
  • • Нажмите на сервер и кнопку подключения.
02. iOS / iPhone

Рекомендуемое приложение: V2Box

  • • Скопируйте VLESS ключ из бота.
  • • Перейдите во вкладку Configs.
  • • Нажмите Add Config -> Import from clipboard.
  • • Выберите Blood и включите соединение.
03. PC / Windows

Рекомендуемое приложение: v2rayN

  • • Скачайте v2rayN с GitHub.
  • • Нажмите Servers в верхнем меню.
  • • Выберите "Import bulk URL from clipboard".
  • • Нажмите Enter и активируйте сервер.

ПОЧЕМУ BLOOD VLESS?

В отличие от обычных VPN, наш сервис использует технологию Reality. Это делает ваше соединение похожим на посещение обычного сайта. Провайдер не видит, что вы используете обход блокировок. Это гарантирует работу даже при самых жестких блокировках интернета.
